Политика конфиденциальности

Дата последнего обновления: 16 декабря 2025 г.

Собираемые данные

Бот сохраняет историю переписки (текст, фото, медиа), настройки и публичную информацию об участниках (username, имя, время обращения) для поддержания контекста диалога и аналитики работы.

Использование данных

Собранные данные применяются исключительно для обеспечения непрерывности диалога, генерации игр и улучшения качества ответов.

Хранение и безопасность

Данные хранятся в удалённой базе данных Redis с применением следующих мер защиты:

  • Шифрование: Персональные данные (имена, username) шифруются алгоритмом AES-256 (Fernet)
  • Псевдонимизация: Идентификаторы пользователей хэшируются с использованием HMAC-SHA256
  • Минимизация: Хранятся только необходимые для работы данные
  • Защита доступа: Используется TLS-шифрование соединения с базой данных

Удаление ваших данных

Вы можете удалить все сведения о себе:

  • В личных чатах: отправьте команду /delete_data.
  • В групповых чатах: команду /delete_data могут использовать только администраторы группы.

После удаления вся история и настройки для чата стираются безвозвратно.

Сторонние сервисы

Для генерации ответов ваши сообщения могут обрабатываться через API одного из следующих сервисов:

  • Google Gemini: Основной провайдер для обработки текстовых и мультимодальных запросов
  • OpenRouter: Сервис-агрегатор для доступа к различным языковым моделям
  • Pollinations: Сервис для генерации изображений и текста (https://pollinations.ai)

Бот автоматически выбирает наилучший доступный сервис для обработки вашего запроса.
